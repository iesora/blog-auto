WordPress記事を、
AIが書いて自動で公開するサービスです。

👩‍💻

副業でブログ運営する個人の方

仕事の合間に記事を書き続けるのは大変。AIに書いてもらって、コツコツ更新を続けたい方に。

自社メディアを運営する中小企業

集客のためにブログを書いているけれど、月20本のノルマがきつい。コストを抑えて量を出したい方に。

💡解決できるお悩み

これまで「人手でやっていた」ブログ運用が、こんなふうに変わります。

これまで(人手で運用)
  • 1記事書くのに2〜3時間かかる
  • 外注すると1本5,000〜10,000円
  • ライターによって文体がバラバラ
  • 更新が止まると検索順位も落ちる
  • 誤字脱字・NG表現のチェックが大変
→
Blog Auto を使うと
  • 1記事あたりわずか数十円〜
  • キーワードを登録するだけ

  • 夜や休日も自動で更新が続く
  • NGワードは事前に自動チェック

✨主な機能

「やりたいこと」ごとに、どんな機能が用意されているかをご紹介します。

🔌

WordPressサイトとの接続 最初に1回だけ

ご利用中のWordPressサイトを、Blog Auto に登録します。サイト側に追加のソフトを入れる必要はありません。

具体的にできること
  • サイトのURLとパスワードを入力するだけで接続完了
  • 接続できているか自動でチェック
  • WordPressのカテゴリ・タグを自動で取り込み
📅

記事の予約投稿 毎回の作業

キーワードと公開したい日時を登録すると、その時刻にあわせてAIが記事を生成し、自動でWordPressに投稿します。

登録するもの
  • 記事のキーワード(例:「副業 確定申告」)
  • 記事のタイプ(解説 / ハウツー / 比較 など)
  • 公開したい日時(5分単位で指定可能)
  • サムネイル画像を作るかどうか
  • 文字数の目安(2000 / 4000 / 6000文字)
📋

まとめて予約(CSV取り込み) 大量の更新に

キーワードを表計算ソフトでまとめておけば、ファイルをアップロードするだけで一気に何十本も予約できます。

こんなときに便利
  • 1ヶ月分の記事をまとめて仕込んでおきたい
  • SEO担当者がExcelで作ったキーワードリストをそのまま使いたい
🎨

サイトごとのトーン設定 ブランド統一

「ですます調で書く」「NGワードは避ける」「最後に決まったCTAを入れる」など、サイトのカラーをあらかじめ設定できます。

設定できる内容
  • 業種・読者の年代や知識レベル
  • 文体(です・ます調 / である調)
  • 使ってはいけないNGワード
  • 必ず記事の最後に入れたい文章(CTA)
  • 薬機法・景表法に触れそうな表現の自動チェック
🖼️

記事画像・サムネイルの自動生成 画像も自動

記事に合わせたサムネイル画像を、AIが自動でつくります。記事の内容に合った雰囲気の画像が生成されるので、わざわざ写真素材を探す必要がありません。

📊

使用量ダッシュボード 使いすぎ防止

今月どれくらい記事を生成したか、どれくらいコストがかかっているかが、ひと目でわかります。プラン上限を超えそうになったら自動でお知らせします。

暴走を防ぐしくみ
  • プラン上限に達したら自動で生成停止
  • 1日あたりのAI利用料に上限を設定可能
  • 想定の1.5倍を超えると自動でブレーキ
👥

チームでの利用・権限管理 複数人で運用

1つのアカウントに、複数のメンバーを招待できます。役割によって「見るだけ」「記事だけ書ける」「すべて管理できる」など権限を分けられます。

4つの役割
  • オーナー:すべての操作と請求情報の管理
  • 管理者:メンバー招待やサイト編集
  • 編集者:記事の生成と予約
  • 閲覧者:結果を見るだけ
💳

かんたんお支払い セルフサービス

クレジットカードを登録するだけで、毎月自動で請求されます。プラン変更・解約・カード変更・請求書ダウンロードもすべて画面から操作できます。

📜

操作ログ(監査ログ) 企業の安心

「いつ・誰が・何をしたか」を自動で記録します。法務やコンプライアンス対応が必要な企業様にも安心してご利用いただけます。

🤖AIモデルと1記事のコスト

記事を書くAIには「賢さ」と「料金」が違うモデルがいくつかあります。サイトの用途や予算にあわせて、好きなモデルを選べます。

かんたんに言うと: AIにも「軽自動車」「セダン」「高級車」のような違いがあります。スピード重視か、品質重視かで使い分けます。サムネ画像にもAIを使っており、こちらも別のモデルを選べます。

3つのモードから選べます

🌱

節約モード 大量更新向け

もっとも軽くて速いAIを使います。1記事あたり10〜20円ほどで、月100本でも合計2,000円程度。とにかく更新頻度を上げたいときに。

使われるAIモデル
  • 本文:Claude Haiku 4.5(軽量・高速モデル)
  • 画像:Gemini Flash(軽量モデル)
  • 品質:標準的な記事は十分書ける。専門的な内容はやや弱い。
⚖️

バランスモード おすすめ・初期設定

品質とコストのバランスがいい中堅モデル。SEO狙いの本格ブログにちょうどよく、ほとんどのお客様はこのモードを使っています。

使われるAIモデル
  • 本文:Claude Sonnet 4.6(中堅モデル)
  • 画像:Gemini 2.5 Pro
  • 品質:文章のまとまり・読みやすさが高い。多くの分野で活用可能。
💎

高品質モード 専門記事向け

もっとも賢いAIを使います。1記事に100〜180円ほどかかりますが、専門性の高い記事や、社外に出しても恥ずかしくない高品質な記事を書くときに。

使われるAIモデル
  • 本文:Claude Opus 4.7(最上位モデル)
  • 画像:Gemini 2.5 Pro
  • 品質:複雑なテーマ・専門分野でも自然な文章。長い記事も得意。

1記事あたりのコスト比較

1記事 = 約4,000文字 + サムネイル画像1枚を生成した場合の目安です(2026年5月時点)。

コストに影響する3つのポイント

📏

① 記事の長さ

文字数が多いほどコストは増えます。2000文字なら半額、6000文字なら1.5倍ほどが目安です。

🖼️

② 画像を作るかどうか

サムネイル画像の生成は1枚あたり数円〜十数円。画像なしに設定すれば、そのぶん節約できます。

🎨

③ ブランド設定の量

NGワード・読者像・文体ルールなどを細かく設定するほど、AIに渡す情報が増え、わずかにコストが上がります。

🛡️

④ 暴走防止のブレーキ

1日あたりの上限を設定できるので、想定外の請求は発生しません。上限に達したら自動でストップします。

予算の決め方の例
「月3万円までで、なるべく多く更新したい」場合

バランスモード(1記事¥40前後)であれば、月750本ほど作れる計算です。
「専門記事を月50本だけ」なら、高品質モード(1記事¥150前後)で月¥7,500ほどの予算になります。
実際の運用では、メインは節約モード/重要記事だけ高品質モード、といった記事ごとのモード切り替えもできます。

ご注意: 上記の金額はAI各社の料金体系をもとにした目安です。実際の請求額は記事の文字数・プロンプトの長さ・生成回数によって変動します。料金が改定された場合は、改めてご案内します。

🛡️安心のしくみ

大切なサイトの情報をお預かりするので、セキュリティと運用には特に気を配っています。

🔐

パスワードは暗号化して保管

WordPressのパスワードや各種情報は、すべて暗号化された状態でお預かりします。万が一データが流出しても、中身は読み取れません。

たとえると: 銀行の金庫にしまうイメージ。鍵を持っているのは弊社のシステムだけ。
🚧

他のお客様の情報とは完全に分離

お客様ごとに別の引き出しに情報が入っているイメージです。間違って他社の情報が見えてしまうことが起きないよう、システムの仕組みで強制しています。

テスト: 開発時に「他人のデータが見えないか?」のチェックを毎回自動で実施。
📡

通信はすべて暗号化

お客様と弊社サーバーの間のやり取りは、すべて暗号化されています。途中で情報を盗み見られる心配はありません。

⚡

使いすぎ防止のブレーキ

AIの利用料が思った以上にかかってしまわないよう、1日あたりの上限を設定。想定を超えたら自動でストップします。

💾

毎日のバックアップ

データベースは毎日自動でバックアップ。30日前までさかのぼって復元できる仕組みです。

🔔

24時間の自動監視

システムに異常があれば、すぐに担当者へ通知が届きます。お客様より先に弊社が気づき、対応します。

🗺️これからの予定

サービスは3段階に分けて、できることを少しずつ増やしていきます。

まず最初に提供(0〜3ヶ月)

基本機能の提供

会員登録・WordPress接続・記事の自動生成と公開・月額課金など、サービスとして使い始めるために必要な機能を一通りご用意します。

次の段階(4〜6ヶ月)

複数サイト・チーム対応の強化

複数サイトの一括管理、メンバーの役割管理、サイトごとの文体細かい設定、使用量グラフ、Google連携、二段階認証など、ビジネスでの活用に必要な機能を追加します。

大企業向け(7〜12ヶ月)

エンタープライズ対応

会社のシングルサインオンとの連携、専用環境のご提供、独自AIキーの持ち込み、外部ツール(Zapier・Shopifyなど)との連携、セキュリティ認証(ISMS / SOC2)の取得などを進めていきます。

❓よくある質問

WordPressにプラグインを入れる必要はありますか?
不要です。WordPressが標準でもっている機能(REST APIとアプリケーションパスワード)を使うので、サイト側に新しいプラグインを入れる必要はありません。WordPress 5.6以降であればすぐに使えます。
生成された記事は、自分のサイトで自由に使えますか?
はい、生成された記事の権利はお客様に帰属します。ただし、医療・薬機法・金融など内容にデリケートさが求められる分野では、念のため人の目でチェックしてから公開することをおすすめします。
記事数が上限を超えたら、どうなりますか?
その時点で記事の自動生成が一時停止します。Proプランは「1記事あたり+80円」で続けるか、上位プランへ切り替えていただくか選べます。Starterプランは翌月のリセットまでお待ちいただくか、プランをアップグレードしていただきます。
意図せず大量に課金されることはありません。
AIが書いた記事の品質は大丈夫ですか?
サイトごとの文体・読者層・NGワードを事前に設定できるため、貴社サイトのトーンに合った記事を生成します。気になる場合は「下書き」として保存し、人がチェックしてから公開する運用も可能です。
AIモデルは途中で変更できますか?
はい、いつでも変更できます。サイト単位や記事単位で「節約モード」「バランスモード」「高品質モード」を切り替えられるので、たとえば「メインは節約モードで運用、重要記事だけ高品質モードで生成」といった使い分けも可能です。
解約したいときは?
管理画面からいつでも解約できます。解約後30日間はデータを保持し、その間であれば復元可能です。30日を過ぎると完全に削除されます。GDPRや個人情報保護法に基づくデータ削除のご依頼にも対応します。
サポートはありますか?
Starterプラン以上で、メールサポート(営業日24時間以内に返信)をご提供します。Agencyプラン以上は優先サポート、Enterpriseプランは専用チャンネル+専任担当者をご用意します。
会社のセキュリティ要件が厳しいのですが、対応できますか?
Enterpriseプランでは、会社のシングルサインオンとの連携、専用環境のご提供、SLA保証、各種セキュリティ認証への対応などをご相談できます。まずは個別にご要件をお聞かせください。
自社のAPIキーを使うことはできますか?
はい、Phase 3でご提供予定です(BYOK機能)。自社で契約しているAnthropicやGoogleのAIキーを持ち込んでいただければ、AI利用料を自社契約側に切り替えることができます。Enterpriseプランで先行ご提供します。
