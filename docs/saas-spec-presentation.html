SaaSを自社で作るのは、想像以上に重い。

「AIを使った新規事業を立ち上げたい」「既存ツールをSaaS化したい」
そう思っても、社内開発で詰まりやすい3つのポイントを、弊社が肩代わりします。

🧩

SaaS基盤のゼロ実装が大変

マルチテナント・課金・キュー・監査ログ・KMS暗号化。要件整理だけで数ヶ月、実装で半年は珍しくありません。

🤖

AI統合の知見が社内にない

Claude / Gemini / プロンプト設計 / トークン課金管理 / 暴走防止のサーキットブレーカ。LLM活用には独自のノウハウが必要です。

📈

運用後のスケールが読めない

テナント数が増えたとき、コストはどう増える? SLAをどう守る? 当初の設計次第で、運用コストは数倍変わります。

SCOPE

SaaSとして必要なものは、
標準スコープに含めます。

マルチテナント基盤・課金・キュー・監査ログ・セキュリティ — エンタープライズ要件まで考え抜いた設計。

🏢

マルチテナント設計

Organization単位でデータを完全分離。ORM層に tenant_id スコープを強制し、漏洩リスクを構造的に排除します。

🔐

KMS暗号化

WordPressアプリケーションパスワード・APIキーはGCP Cloud KMSで暗号化。90日ごとのDEK自動ローテーション。

💳

Stripe従量課金

月額サブスク + 記事数のMetered Billing。Stripe Customer Portalでカード変更・請求書DLまでセルフサービス化。

⚙️

非同期ジョブキュー

Cloud Tasks ベースの実行基盤。テナント別concurrency制御・指数バックオフ・DLQで失敗時もリトライ。

📊

使用量メータリング

記事/画像生成数をリアルタイム計測。プラン上限・日次コスト上限・サーキットブレーカで暴走を防止。

🎨

BrandProfile

業種・読者ペルソナ・文体・NGワード・必須CTAをテナント別に設定。few-shotで過去の高評価記事を学習。

🔌

公開API + Webhook

REST API・HMAC署名つきWebhook。ZapierやMakeとの連携も視野。Phase2以降で実装可能。

👥

チーム・RBAC

Owner / Admin / Editor / Viewer の4階層ロール。代理店プランで複数クライアントを安全に管理。

📋

監査ログ

全書き込み操作を audit_logs に記録。Owner権限で90日分閲覧可能。法務・コンプラ対応にも。

FAQ

外注先選定でよくいただくご質問

ソースコードは納品されますか? 知的財産は誰のものになりますか?

完全納品いたします。納品物の著作権・知的財産権はすべて貴社へ譲渡される契約形態を標準としています。GitHubリポジトリの管理権限ごと移管し、弊社側のアクセスは契約終了時点で削除します。OSSライブラリのライセンス管理一覧もあわせてお渡しします。

納期はどれくらいですか? 遅延した場合の対応は?

MVPプランで3ヶ月、SaaSローンチプランで5〜6ヶ月が標準です。週次スプリント + デモで進捗を可視化し、遅延リスクは早期に共有します。マイルストーン未達時のペナルティ条項を契約に明記することも可能です。

開発中の機密情報はどう守られますか?

契約前のヒアリング段階からNDAを締結します。開発中のソースは貴社GitHub Organization内のプライベートリポジトリで管理し、アクセス権を限定。本番のAPIキー・顧客データには弊社開発メンバーは原則アクセスしません。

納品後、自社で運用できますか? 引き継ぎはどうなりますか?

納品物にはコードのほか、アーキテクチャ図・運用マニュアル・オンコール手順書・トラブルシューティング集が含まれます。引き継ぎセッション(計4〜6時間想定)を実施し、納品後3ヶ月間は質問対応窓口を無料でご提供します。自社運用が不安な場合は月額¥15万〜の保守プランもご利用いただけます。

AIのAPI費用やGCPのインフラ費用は誰が負担しますか?

原則として貴社契約のアカウントで運用します(Anthropic・Google・Stripe・GCP)。開発期間中の検証費用も貴社負担となりますが、初期見積もり時にコスト試算をお渡しします。BYOK構成のためAPIコストの透明性は最大化されます。

追加機能の依頼や仕様変更はできますか?

可能です。Phase1スコープ内の仕様調整は当初見積もりに含みます。スコープ外の追加機能は、別途お見積もりのうえ着手します。納品後のグロース支援(Phase2/3)も同じ体制で継続対応できます。

過去にどんな開発実績がありますか?

NestJS + Cloud Run構成でのマルチテナントSaaS、AI(Claude / Gemini)を活用したコンテンツ生成基盤、Stripe従量課金の本番運用などの実績があります。具体的な実績はNDA締結後に個別にご紹介可能です。

請求・契約形態はどうなっていますか?

請負契約を標準とします。着手金30% / 中間40% / 検収30%のマイルストーン分割が一般的です。準委任(SES型)・継続的な保守委託のいずれにも対応可能です。請求書払い・分割払いもご相談ください。
