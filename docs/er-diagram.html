テーブル概要

sites

マルチサイト管理。WordPress 認証情報 (暗号化) と GSC サイトURL、デフォルトの記事タイプ・カテゴリ・タグ・ペルソナを保持。

schedule_entries

サイト×日付の投稿スケジュール。キーワード3本・トピック・記事タイプ等を持ち、KeywordPlan 起点 (auto) か手動 (manual) の二系統。

run_history

各スケジュールの実行履歴。成功/失敗、WordPress 投稿 ID/URL/タイトル、エラー、所要時間を記録。

keyword_plans

サイト単位のキーワード計画 (サイクル期間)。GSC スナップショットを根拠に LLM が生成、承認フローあり。

gsc_snapshots

Search Console データの取得スナップショット。期間と行数、データ確定状態を保持。

gsc_query_rows

スナップショット配下のクエリ行。query/page/clicks/impressions/ctr/position。impressions に複合インデックス。
